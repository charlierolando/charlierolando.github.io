This project is part of the State Polytechnic of Batam Internal Research Program 2023. This project aims to develop system to automate repetitive tasks, such as moving objects from one place to another, placing parts, or other processes that require highly precision, accurate and efficiency.

My task is to writing, testing, debugging, and maintaining the Ladder code to create robot motion.
